# vision-comms-zmq

## Install requirements

```
pip install pyzmq
```
or
```
pip install zmq
```


## Run the zmq example


Run the publisher that generates bboxes
```
python pub_bboxes.py 
```

Run the subscriber:
```
python sub_bboxes.py
```
